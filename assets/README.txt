HOW TO ADD YOUR LOGO
====================

Save your logo image file in THIS folder with the exact name:

    logo.png

(A transparent PNG or a plain image both work. Ideal height ~ 96px or larger,
so it stays crisp on high-resolution screens.)

That's it. As soon as "assets/logo.png" exists, the website header will
automatically show your real logo instead of the built-in placeholder.
No code changes are needed.

Optional: a light/white version of the logo for the dark footer can be added
as "logo-white.png" — tell the developer if you'd like it wired into the footer.
